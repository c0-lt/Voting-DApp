# Voting Dapp Project

## Installation

After cloning the project, run the following commands. Make sure ganache is running before.

```sh
# Deploy the contract locally
$ cd truffle
$ npm install
$ truffle migrate --reset
```

```sh
$ cd client
$ npm start
  Starting the development server...
```
